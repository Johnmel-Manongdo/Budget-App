# Budget App

A mobile application for budget tracking.

## Features

- Account Login
- Add and Edit expense or income
- View transaction history
- View statistics
- Set period (monthly, weekly)
- Set limit for amount
- Change currency
- Add categories
- Calculator
- Calendar
